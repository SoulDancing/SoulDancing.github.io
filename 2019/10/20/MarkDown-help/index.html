MarkDown 语法

MarkDown是一种纯文本格式的标记语言。通过简单的标记语法,它可以使普通文本内容具有一定的格式。

2. 斜体和粗体

  • 常规体

  • 斜体 or 斜体

  • 粗体

  • 斜体加粗体

3. 超链接

  • 行内式 : 超链接

  • 自动链接 : https://xxx

  • 参考式 : 我经常去的几个网站Google、Leanote以及自己的博客
